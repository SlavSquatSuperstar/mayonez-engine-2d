# MayonezEngine
 The 2D Java game engine to fit your Slavic needs!*†
 
 Java Version: JDK/JRE 11

 Dependencies (bundled in JAR)
 - JSON in Java (org.json)
 - Apache Commons Lang
 - Apache Commons IO
